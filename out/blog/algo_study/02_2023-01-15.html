자바스크립트의 맵(Map)과 셋(Set)

맵(Map)

맵은 키와 데이터를 같이 저장하는 자료구조이다. 자바스크립트에서의 맵은 아래와 같은 메서드와 프로퍼티를 가지고 있다.

const map = new Map(); // 맵 생성

map.set(key, value); // key를 이용해 value 저장
map.get(key); // key에 해당하는 value 반환 (없으면 undefined)
map.has(key); // key가 존재하면 true, 아니면 false 반환
map.delete(key); // key에 해당하는 값 삭제
map.clear(); // map 안의 모든 요소 삭제
map.size; // map의 요소 개수 반환

map.keys(); // 각 요소의 key를 모은 이터러블 객체 반환
map.values(); // 각 요소의 value를 모은 이터러블 객체 반환
map.entries(); // 각 요소의 [key, value]를 한 쌍으로 하는 이터러블 객체 반환

Object.entries(obj); // object를 map으로 변환
Object.fromEntries(map); // map을 object로 변환

맵은 키에 다양한 자료형을 허용하고, 값이 삽입된 순서대로 저장된다는 점에서 객체와 차이점이 있다.

셋(Set)

셋(Set)은 중복을 허용하지 않는 데이터들의 집합인 자료구조이다. 자바스크립트에서의 셋은 아래와 같은 메서드와 프로퍼티를 가지고 있다.

const set = new Set(iterable); // 셋 생성

set.add(value); // value를 저장하고 셋 자신을 반환
set.delete(value); // value를 삭제하고 성공하면 true, 아니면 false 반환
set.has(value); // value가 존재하면 true, 아니면 false 반환
set.clear(); // set 안의 모든 요소 삭제
set.size; // set의 요소 개수 반환

set.keys(); // set의 모든 값을 포함하는 이터러블 객체 반환
set.values(); // set의 모든 값을 포함하는 이터러블 객체 반환
set.entries(); // set의 모든 값을 [value, value] 쌍으로 가지고있는 이터러블 객체 반환
