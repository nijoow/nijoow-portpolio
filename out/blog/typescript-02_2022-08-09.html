타입스크립트의 타입은?

타입스크립트는 일반 변수, 매개 변수, 객체 속성 등에 : TYPE과 같은 형태로 명시적으로 타입을 지정할 수 있습니다. 타입스크립트는 자바스크립트의 타입을 그대로 사용하며, 타입스크립트의 고유 타입이 추가로 제공됩니다.

  • boolean
  • number
  • string
  • array
  • object
  • function
  • void
  • tuple
  • enum
  • any
  • null
  • undefined
  • never

boolean

논리형은 true와 false로 진위값을 나타냅니다.

let bool: boolean = true;

number

숫자형은 정수, 실수, Infinity, NaN 등 숫자값을 나타냅니다.

let num: number = 7;

string

문자열은 ES6의 템플릿 문자열도 지원합니다.

let text: string = "text";
let str: string = `this is ${text}`;

array

배열은 두가지 방법으로 타입을 선언할 수 있습니다.

// 특정 타입의 데이터만 가지는 배열
let strArr: string[] = ["A", "B", "C"];
let strArr: Array<string> = ["A", "B", "C"];

// Union(다중) 타입의 데이터를 가지는 배열
let unionArr: (string | number)[] = ["A", 1, "B", 2];
let unionArr: Array<string | number> = ["A", 1, "B", 2];

// 모든 타입의 데이터를 가질 수 있는 배열
let anyArr: any[] = ["A", 1, true];
let anyArr: Array<any> = ["A", 1, true];

object

객체형은 typeof 연산자가 object로 반환하는 모든 타입을 나타냅니다.

let obj: object = {};
let arr: object = [];
let func: object = function () {};
let date: object = new Date();

정확한 타입 지정을 위해서 객체 속성들에 대한 타입을 개별적으로 지정할 수 있습니다.

let obj: { str: string; num: number } = {
  str: "text",
  num: 12,
};

function / void

함수는 매개 변수에 각각의 타입을 지정해줘야합니다. 그리고 리턴값이 있다면 명시적으로 반환 값의 타입을 지정할 수 있습니다.

function sum(a: number, b: number): number {
  return a + b;
}

리턴 값을 반환하지 않는 함수의 경우에 void 타입을 지정합니다.

function func(): void {
  console.log("no return");
}

tuple

튜플 타입은 배열 타입을 특수한 형태로 사용할 수 있는 타입입니다. 배열 안의 요소들의 개수 만큼 명시적으로 지정하고 순서나 개수가 다르면 오류가 발생합니다.

let tuple: [string, number] = ["text", 7];

tuple = [7, "text"]; // error
tuple = ["text", 7, 12]; // error

enum

열거형은 멤버라 불리는 명명된 값의 집합을 이루는 자료형입니다. 멤버는 기본적으로 0부터 시작하고 값은 1씩 증가합니다.

enum obj {
  a,
  b,
  c,
  d,
}
// obj.a => 0   obj[0] => a
// obj.b => 1   obj[1] => b
// obj.c => 2   obj[2] => c
// obj.d => 3   obj[3] => d

아이템에 값을 할당할 수도 있습니다. 할당되지 않은 아이템은 이전 아이템의 값에 +1이 된 값이 설정됩니다.

enum obj {
  a,
  b = 50,
  c,
  d = 100,
}
// obj.a => 0   obj[0] => a
// obj.b => 50   obj[50] => b
// obj.c => 51   obj[51] => c
// obj.d => 100   obj[100] => d

any

any는 모든 타입을 허용한다는 것을 의미합니다. 자바스크립트를 타입스크립트로 적용시키는 과정이나 불가피하게 타입을 선언하기 힙든 경우(외부 라이브러리 사용 등)에 유용할 수 있습니다. 하지만 타입스크립트의 의미와 반대되고 예상치 못한 오류를 만들어낼 가능성이 높아지기 때문에 최대한 지양해야하는 타입입니다.

let any: any = "text";
any = 0;
any = true;
any = {};

null / undefined

null 은 값이 없다는 것을 명시해주는 데이터 타입이자 하나의 값이고, undefined는 값이 할당되지 않은 변수를 참조할 때, 객체의 정의되지 않은 속성에 접근하면 리턴되는 값입니다.

never

never는 절대 발생할 수 없는 타입을 나타냅니다. 일반적으로 함수의 리턴 타입으로 사용되고, 항상 오류를 출력하거나 무한 루프로 리턴값을 내보내지 않음을 의미합니다.

function errorMessage(): never {
  throw new Error("error!");
}
function infiniteLoop(): never {
  while (true) {
    infiniteLoop();
  }
}
